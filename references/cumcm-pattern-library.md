# CUMCM 赛题套路识别库

## 十二种核心套路

---

### 套路 1：预测→优化

**特征**：先预测关键参数，再基于预测结果做优化决策。

**真题示例**：
- 2023B 多波束测深：预测海底地形 → 优化测线间距
- 2021B 乙醇偶合制备：预测C4烯烃收率 → 优化反应条件
- 2022B 无人机巡航：预测风速场 → 优化飞行路径

**典型模型链**：
- LSTM/ARIMA 预测 → MILP/NLP 优化
- XGBoost 预测 → NSGA-II 多目标优化
- Prophet 预测 → 鲁棒优化

**国一创新热点**：
- 预测不确定性→随机优化
- 多步预测→滚动优化
- 注意力机制改进预测精度

---

### 套路 2：评价→预测

**特征**：先建立评价体系打分/分类，再预测未来趋势。

**真题示例**：
- 2020C 中小微企业信贷决策：评价企业信用 → 预测违约概率 → 制定信贷策略
- 2015B 互联网+出租车：评价供需匹配度 → 预测打车难度时空分布

**典型模型链**：
- 熵权TOPSIS 评价 → BP神经网络/LSTM 预测
- AHP模糊综合评价 → XGBoost 预测
- CRITIC+VIKOR → 时序预测

**国一创新热点**：
- 评价结果作为特征输入预测模型
- 动态权重评价体系
- 反馈式评价→预测→再评价

---

### 套路 3：运筹优化

**特征**：纯优化问题，目标函数+多约束，可能多目标。

**真题示例**：
- 2022B 无人机巡航：飞行路径优化（时间最短/覆盖最大）
- 2021C 生产企业原材订购：订购策略优化（成本最低/供应最稳）
- 2020B 穿越沙漠：路径规划多目标优化

**典型模型链**：
- 混合整数规划(MILP) → 灵敏度分析
- NSGA-II 多目标 → TOPSIS 方案优选
- 遗传算法 → 模拟退火精修

**国一创新热点**：
- 实际约束引入（法规/物理极限/经济条件）
- 鲁棒优化应对不确定性
- 多阶段动态规划替代静态优化

---

### 套路 4：微分方程

**特征**：物理/工程系统，微分/偏微分方程建模。

**真题示例**：
- 2019A 高压油管控制：流体动力学 PDE
- 2018A 高温作业服装：热传导 PDE
- 2016A 系泊系统设计：力学 ODE

**典型模型链**：
- 物理定律 → ODE/PDE建立 → 有限差分求解 → 参数标定 → 敏感性分析
- 机理模型 → 有限元 → 最小二乘拟合参数
- PDE → PINN求解（参数未知场景）

**国一创新热点**：
- 多物理场耦合
- 非线性项精确处理
- 参数反演方法创新

---

### 套路 5：评价体系构建

**特征**：多指标综合评价，权重确定和排序方法是核心。

**真题示例**：
- 2016B 小区开放对道路通行影响：建立通行能力评价体系
- 2014B 创意平板折叠桌：评价设计方案的优劣

**典型模型链**：
- 指标筛选（相关分析/变异系数）→ 赋权（熵权法/CRITIC/AHP）→ 排序（TOPSIS/灰色关联）
- AHP层次分解 → 模糊综合评价
- 主客观组合赋权 → TOPSIS/VIKOR

**国一创新热点**：
- 组合赋权（主客观融合）
- 动态权重
- 敏感性分析验证权重稳定性

---

### 套路 6：分类+特征提取

**特征**：先对样本分类，再提取每类特征规律。

**真题示例**：
- 2017B 拍照赚钱任务定价：分类任务类型 → 提取定价特征
- 2022C 古代玻璃成分分析：聚类玻璃类型 → 提取成分特征 → 判别来源

**典型模型链**：
- K-means/层次聚类 → 特征分析 → 决策建议
- PCA降维 → SVM/RF分类 → 特征重要度分析
- DBSCAN去噪 → GMM软聚类 → 每类独立建模

**国一创新热点**：
- 无监督→半监督→有监督递进
- 聚类结果作为新特征
- 异常检测+正常模式建模

---

### 套路 7：仿真建模

**特征**：物理/工程系统仿真，多参数交互。

**真题示例**：
- 2016A 系泊系统设计：钢桶/重物球/锚链力学仿真
- 2014A 嫦娥三号软着陆：轨道仿真

**典型模型链**：
- 机理建模 → 蒙特卡洛仿真 → 敏感性分析 → 最优参数确定
- Agent-Based建模 → 涌现现象分析 → 参数调优
- 系统动力学(SD) → 反馈回路分析 → 情境模拟

**国一创新热点**：
- 多精度仿真（粗→细递进）
- 仿真+优化混合
- 不确定性量化

---

### 套路 8：图论与网络

**特征**：路径规划/网络流/最短路径/最小生成树。

**真题示例**：
- 2015A 太阳影子定位：图论建模地理位置关系
- 2021C 生产企业运输：运输网络优化

**典型模型链**：
- 图构建 → 最短路径(Dijkstra/A*) → 路径优化
- 网络流建模 → 最大流/最小费用流
- PageRank → 关键节点识别 → 网络优化

**国一创新热点**：
- 动态图（时变网络）
- 多层网络
- 图神经网络辅助

---

### 套路 9：时间序列预测

**特征**：时间序列数据，趋势+周期+噪声分解。

**真题示例**：
- 2020A 炉温曲线：焊接温度时序预测
- 2017A CT系统参数标定：投影数据时序分析

**典型模型链**：
- 时序分解(STL/EMD) → 各分量独立预测 → 重构融合
- SARIMAX → 残差LSTM修正
- Prophet基线 → XGBoost/Informer提升

**国一创新热点**：
- 多尺度分解（EMD/小波）
- 变点检测+分段建模
- 外部协变量引入

---

### 套路 10：统计建模

**特征**：概率统计模型，参数估计，假设检验。

**真题示例**：
- 2015C 月上柳梢头：天文统计建模
- 2019C 机场出租车决策：概率统计决策

**典型模型链**：
- 分布假设 → 参数估计(MLE/贝叶斯) → 假设检验
- 概率模型 → 蒙特卡洛模拟 → 置信区间

**国一创新热点**：
- 贝叶斯层次模型
- 混合分布建模
- Bootstrap不确定性量化

---

### 套路 11：机器学习综合

**特征**：大规模数据，特征工程+模型集成。

**真题示例**：
- 2023C 电商评论分析：NLP+分类+聚类
- 2018C 大型百货商场会员画像：特征工程+聚类+关联规则

**典型模型链**：
- 数据清洗 → 特征工程(构造/选择/降维) → 集成学习(Stacking/Blending) → 结果解释
- NLP预处理 → 词嵌入 → LSTM/Transformer分类
- 关联规则(Apriori/FP-Growth) → 序列模式挖掘

**国一创新热点**：
- 自动化特征工程
- 多模态特征融合
- 模型可解释性(SHAP/LIME)

---

### 套路 12：多阶段耦合

**特征**：多个子问题之间存在反馈/耦合，不是独立求解。

**真题示例**：
- 2020B 穿越沙漠：阶段决策影响后续状态
- 2022A 波浪能装置：多个物理子系统耦合

**典型模型链**：
- 子系统独立建模 → 耦合接口定义 → 迭代联合求解
- 粗粒度初解 → 细粒度精修 → 全局一致性回代
- 动态规划 → 多阶段决策 → 滚动优化

**国一创新热点**：
- 耦合变量设计
- 多层迭代收敛策略
- 子系统解耦降维
