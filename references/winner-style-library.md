# 国一论文风格学习库

---

## 一、摘要风格（3 种经典模板）

### 模板 A：问题穿透型

**适用**：物理/工程类题目，问题自身逻辑链清晰。

**结构**：
```
[问题本质+难点一句话] →
[核心视角/工具切入] →
[模型体系，按逻辑关系而非罗列] →
[关键结果，有对比基准] →
[结论延伸价值]
```

**示例**：
> "针对焊接过程中温度场精确控制这一典型热传导反问题，本文从'正演-反演-验证'三位一体的视角切入，建立了有限差分正演→最小二乘参数反演→蒙特卡洛不确定性量化的完整求解链路。在参数反演环节，引入对流传热修正项替代标准辐射假设，使温度预测均方根误差从7.2%降至2.1%。基于反演结果的正演预测与实测值偏差<3%，验证了模型的可靠性。该研究为焊接工艺参数的在线优化提供了可量化的温度预测工具。"

**真题示例**：2019A 高压油管控制、2018A 高温作业服装

---

### 模板 B：数据驱动型

**适用**：大数据类题目，机器学习/统计方法为主。

**结构**：
```
[领域+传统方法局限] →
[数据规模+来源] →
[模型体系+创新机制] →
[实证结果，含与基线对比] →
[工程/决策意义]
```

**示例**：
> "中小微企业信贷评估长期依赖专家经验，但123,000条历史信贷数据表明，传统评分卡模型的违约预测准确率仅72%。本文基于该数据集，构建了'多策略缺失值填补→熵权TOPSIS企业画像→XGBoost违约预测→最优信贷策略树'的完整决策链路。创新性地将企业交易稳定性、行业景气度等外源特征融入TOPSIS评价框架，使AUC从0.78提升至0.91。基于模型输出的差异化信贷策略较现行方案预期坏账率降低34%，为银行普惠金融决策提供了量化依据。"

**真题示例**：2020C 中小微企业信贷、2023C 电商评论分析

---

### 模板 C：指标驱动型

**适用**：评价/决策/方案优选类题目。

**结构**：
```
[决策问题定性] →
[评价指标体系概述] →
[赋权+排序方法，强调方法组合的合理性] →
[方案排序/择优结果] →
[方案的实际含义+稳定性验证]
```

**示例**：
> "如何量化评估小区开放对周边道路通行的影响，是城市规划中的典型多指标决策问题。本文从通行效率、安全性和环境影响三个维度构建了包含12个指标的评价体系。针对指标间既存在相关性又存在冲突性的特点，采用CRITIC客观赋权与AHP主观赋权的博弈论组合权重，结合TOPSIS方法对5种开放方案进行排序。结果显示方案3（半开放+潮汐车道）综合最优，且该排序结果在权重±15%扰动下保持稳定（Spearman秩相关系数>0.92），验证了结论的鲁棒性。"

**真题示例**：2016B 小区开放、2014B 创意平板折叠桌

## 二、创新呈现模式（2 种）

### 模式 A：缺陷驱动型（推荐首选）

**逻辑**：
```
标准[模型名]在[任务]上表现出[优点]，然而其[缺陷1]和[缺陷2]
导致在本题[具体场景]下性能退化。

为此，我们提出[创新点]：[具体做了什么]，其核心思想是[一句话概括]。

从数学上，标准模型为：
  [标准公式]
而改进后的模型为：
  [改进公式]
两者的关键区别在于[一句话差异]。
```

**示例**：
> "标准LSTM在处理长时间序列时平等对待每个时间步的隐藏状态，然而本题中焊接温度的关键转折点（峰值温度、冷却速率拐点）包含远高于平稳段的信息量。为此，我们提出注意力加权LSTM：在隐藏状态之上增加一个可学习的注意力层，使模型自动聚焦于关键时间点。标准LSTM的输出为h₁,h₂,...,h_T的等权平均，而改进后为∑αᵢhᵢ，其中αᵢ由式(8)根据当前隐藏状态与全局上下文的相关性自适应计算。"

---

### 模式 B：洞察驱动型

**逻辑**：
```
观察[数据特征/物理规律]发现：[关键发现]。

这一发现启发我们[创新思路]：不同于标准方法[传统做法]，
我们[新的做法]。其合理性在于[一句话物理/数学/统计直觉]。

实验证实了这一点：[量化证据]。
```

**示例**：
> "观察附件1中不同时间段的温度曲线发现：白天的温度波动幅度是夜间的2.3倍，但标准ARIMA模型假设方差恒定。这一发现启发我们：不应使用单一模型处理全天数据，而应按光照强度分段建模。白天时段采用GARCH捕捉异方差，夜间时段采用标准ARIMA。实验证实：分段策略将全天平均预测误差从5.1%降至2.8%，其中白天段改进尤为显著（7.2%→2.9%）。"

## 三、模型评价风格（国一 vs 省二）

### 省二水平（🔴 禁止使用）

| 原句 | 问题 |
|------|------|
| "模型假设较为理想化，未考虑实际中的复杂因素，未来可进一步完善。" | 万能模板，0信息量 |
| "由于时间有限，部分参数未进行精细调整。" | 暴露时间管理失控 |
| "模型具有较好的拟合效果。" | 无量化证据 |
| "本文方法具有一定的参考价值。" | 自我判断，不是事实陈述 |

### 国一水平（✅ 要求达到）

**结构**：

```
[优点的量化证据] → [局限的量化边界] → [P3预判 vs P4实测对比] → [改进的具体路径+前提条件]
```

**示例**：
> "模型在标准工况下表现优异：R²=0.94，残差随机分布于零线两侧（图8），MAE=0.042（较基线ARIMA降低46%）。然而，P3评审曾预判模型在极端温度条件下精度可能退化——P4实测证实了这一预判：当环境温度>40°C时，MAE从0.042膨胀至0.11（图9）。分析原因是高温下材料热物性参数的非线性变化超出了训练数据的覆盖范围。若引入温度依赖的热物性模型（需补充高温段材料参数），预计可将极端工况下的MAE控制在0.06以内。"

## 四、章节节奏分配

| 章节 | 篇幅占比 | 节奏 | 评委关注度 | 写作策略 |
|------|:------:|------|:---------:|----------|
| 摘要 | 5% | 高密度输出，每句都含信息 | ★★★★★ | 最后写，反复打磨，是评委第一印象 |
| 问题重述与分析 | 8% | 快速过，别拖沓 | ★★ | 引用原文+一句话问题结构图描述 |
| 模型假设与符号 | 7% | 中速，论证充分 | ★★★ | 每条假设跟一句"为什么" |
| 模型建立（含创新） | 40% | **慢**，这是高光章节 | ★★★★★ | 核心概念→基础模型→创新展开→子问题递进 |
| 求解与结果 | 20% | 中速，数据密集 | ★★★★ | 算法简述→结果展示→分析差异来源 |
| 模型评价 | 10% | 真诚自解剖 | ★★★★ | 禁用模板句，P3+P4真实张力 |
| 参考文献 | 5% | 格式正确 | ★★★ | 正文中有引用对话 |
| 附录 | 5% | 核心代码片段 | ★★ | 关键算法的核心逻辑 |

## 五、常见扣分句式替换

| 扣分原句（🔴） | 问题 | → 国一替换（✅） |
|------|------|------|
| "我们采用AHP方法确定权重" | 只说做了什么，没解释为什么 | "考虑到指标间的层次性和评委的认知习惯，本文采用AHP将定性判断转化为定量权重，一致性检验结果CR=0.03≪0.1，说明判断矩阵具有满意一致性" |
| "模型具有较好的拟合效果" | 无量化 | "决定系数R²=0.94，且残差随机分布于零线两侧（图3），不存在系统性偏差，说明模型充分捕捉了数据中的结构信息" |
| "本文模型存在一定局限性" | 模板句，无实质 | "如P3评审所预判，模型在[具体条件]下预测精度从0.94降至0.87（降幅7.4%），该场景下的置信区间宽度为±0.12" |
| "通过对参数进行调整，提高了模型精度" | 不说明什么参数、调整幅度 | "学习率从0.01降至0.001后训练稳定，隐藏层节点数经Grid Search确定为64（候选32/64/128），此时验证集MAE达到最低0.042" |
| "在未来研究中，可以考虑更多因素" | 虚无的展望 | "若引入实时温度反馈机制（需部署额外传感器），可将极端工况预测误差从11%压缩至6%（基于P4敏感性外推估计）" |
| "该方法具有推广价值" | 自我表扬 | "该方法的核心——'将物理约束嵌入数据驱动模型'的思路——可迁移至同类反问题，包括热传导参数辨识和扩散系数回归" |
