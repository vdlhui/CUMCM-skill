# CUMCM 模型链模式库（45 条）

## 核心概念

国一论文中，单一模型已罕见。评委期待"模型链"——预处理 → 核心建模 → 后处理/决策的完整链路。

---

## 一、预测链（6 条）

### 链1：降维→时序预测→误差修正
- **结构**：PCA/t-SNE → LSTM/Prophet → ARIMA残差修正
- **适用**：高维时序数据，存在冗余特征
- **真题参考**：2020A 炉温曲线 国一

### 链2：多尺度分解→分量预测→融合重构
- **结构**：EMD/小波分解 → 各分量独立建模 → 加权融合
- **适用**：非平稳序列，多周期性
- **真题参考**：2023B 多波束测深

### 链3：特征工程→集成学习→概率校准
- **结构**：滑动窗统计+频域特征 → XGBoost+LightGBM → Platt Scaling
- **适用**：表格型时序，特征 > 样本
- **真题参考**：2021B 乙醇偶合

### 链4：异常检测→时序预测→异常修正
- **结构**：Isolation Forest → LSTM → 异常点回代修正
- **适用**：含明显异常值的时间序列

### 链5：多粒度分解→独立预测→层次聚合
- **结构**：小波多尺度 → 各尺度独立 ARIMA → 逆小波重构
- **适用**：多频率叠加信号

### 链6：因果推断→关键因子筛选→轻量预测
- **结构**：Granger 因果检验 → 筛选显著因子 → 线性回归
- **适用**：因子过多需降维，样本 < 200

## 二、评价链（9 条）

### 链7：指标筛选→客观赋权→综合评价
- **结构**：相关性分析/变异系数 → CRITIC/熵权法 → TOPSIS/灰色关联
- **适用**：纯数据驱动的评价
- **真题参考**：2016B 小区开放

### 链8：层次分解→主观赋权→模糊评价
- **结构**：AHP 分层 → 判断矩阵 → 模糊综合评价
- **适用**：含定性指标的评价
- **真题参考**：2014B 平板折叠桌

### 链9：聚类分组→组内评价→跨组对比
- **结构**：K-means/层次聚类 → 每组独立评价 → 组间差异分析
- **适用**：样本差异大的评价问题
- **真题参考**：2020C 中小微企业信贷

### 链10：因子分析→主因子评价→综合得分
- **结构**：EFA/CFA → 因子得分 → 综合排名
- **适用**：高维指标(> 15 个)需降维

### 链11：DEA 数据包络→效率评价→投影改进
- **结构**：CCR/BCC 模型 → 效率值排序 → 非有效单元投影分析
- **适用**：多投入多产出效率评价

### 链12：灰色关联→白化权函数→灰色聚类评价
- **结构**：灰色关联度 → 白化权函数 → 灰色聚类
- **适用**：小样本(≤ 30)评价

### 链13：粗糙集→属性约简→规则提取
- **结构**：粗糙集约简 → 核属性 → 决策规则
- **适用**：含冗余定性指标

### 链14：VIKOR→折中排序→优势/劣势/稳定性
- **结构**：群效用+个体遗憾 → 折中排序 → 稳定性验证
- **适用**：决策者偏好不明确的多准则评价

### 链15：云模型→定性定量转换→综合评价
- **结构**：云模型化定性指标 → 云综合 → 云相似度排序
- **适用**：含随机性+模糊性的评价

## 三、优化链（9 条）

### 链16：预测前置→参数确定→优化求解
- **结构**：时序预测关键参数 → LP/NLP/MILP → 灵敏度分析
- **适用**：预测→优化套路
- **真题参考**：2023B 多波束

### 链17：多目标分解→帕累托前沿→决策优选
- **结构**：NSGA-II/MOEA/D → Pareto 解集 → TOPSIS 选最优
- **适用**：多目标冲突
- **真题参考**：2021C 原材料订购

### 链18：可行域搜索→启发式寻优→局部精修
- **结构**：随机采样确定可行域 → GA/PSO → 梯度下降精细调整
- **适用**：非凸/多极值
- **真题参考**：2019A 高压油管

### 链19：双层规划→上下层交替求解→均衡解
- **结构**：上层决策 → 下层响应 → 交替迭代 → Stackelberg 均衡
- **适用**：存在主从博弈关系

### 链20：随机规划→场景生成→期望值优化
- **结构**：拉丁超立方采样 → 场景树/场景缩减 → 期望值模型
- **适用**：含不确定参数的优化

### 链21：鲁棒优化→不确定集→鲁棒解
- **结构**：盒式/椭球不确定集 → 鲁棒对等模型 → 鲁棒解
- **适用**：参数不确定但知道变化范围

### 链22：分布式鲁棒→Wasserstein 距离→分布鲁棒解
- **结构**：经验分布 → Wasserstein 球 → 最坏分布优化
- **适用**：参数分布不确定

### 链23：强化学习→Q-learning→动态最优策略
- **结构**：状态空间/动作空间 → Q-learning/DQN → 最优策略
- **适用**：序贯决策/动态规划

### 链24：博弈论→Nash 均衡→多主体最优策略
- **结构**：博弈要素定义 → Nash 均衡求解 → 稳定性分析
- **适用**：多决策主体利益冲突

## 四、分类链（2 条）

### 链25：特征筛选→降维→分类器
- **结构**：相关性/mRMR → PCA → SVM/Random Forest
- **适用**：高维小样本分类

### 链26：无监督预训练→特征表示→少量标注分类
- **结构**：自编码器/聚类 → 学到的特征 → 半监督分类
- **适用**：标注数据少

---

## 五、仿真链（1 条）

### 链27：机理建模→参数标定→蒙特卡洛验证
- **结构**：ODE/PDE → 数值拟合参数 → MC 不确定度传播
- **适用**：物理过程可建模
- **真题参考**：2019A 高压油管

---

## 六、时空预测链（4 条）

### 链28：空间插值→时序预测→时空融合
- **结构**：克里金/IDW → LSTM/ARIMA → 时空协方差模型
- **适用**：地理空间+时间序列数据

### 链29：图神经网络→时空图→节点预测
- **结构**：GCN/GAT → 时序图构建 → 时空节点预测
- **适用**：路网/电网等网络时空数据

### 链30：地理加权回归→局部预测→空间聚合
- **结构**：GWR(空间变系数) → 局部预测 → 空间模式分析
- **适用**：空间异质性强

### 链31：时空克里金→变异函数→时空最优估计
- **结构**：时空变异函数 → 时空克里金 → 最优无偏估计
- **适用**：稀疏采样点的时空场重建

---

## 七、多源融合链（4 条）

### 链32：异构数据对齐→特征级融合→统一预测
- **结构**：数据标准化/对齐 → 特征拼接/多核 → 统一模型
- **适用**：多来源多格式数据

### 链33：多核学习→核融合→多视角分类
- **结构**：各模态独立核 → 多核融合 → 多视角 SVM
- **适用**：不同模态数据(文本+数值+图像)

### 链34：迁移学习→源域预训练→目标域微调
- **结构**：源域模型训练 → 冻结/微调 → 目标域预测
- **适用**：目标域数据少，有相似源域

### 链35：联邦式→分源独立建模→决策级融合
- **结构**：各源独立建模 → 决策融合(投票/学习权重)
- **适用**：数据不可合并(口径不同/隐私)

## 八、系统动力学链（3 条）

### 链36：因果回路图→存量流量图→SD 仿真
- **结构**：CLD 定性分析 → SFD 定量建模 → Vensim 仿真
- **适用**：多反馈回路复杂系统

### 链37：Agent-Based→微观规则→宏观涌现
- **结构**：Agent 规则定义 → NetLogo/Python ABM → 涌现分析
- **适用**：个体行为驱动系统演化

### 链38：元胞自动机→局部规则→全局模式
- **结构**：CA 邻域+状态规则 → 迭代演化 → 模式识别
- **适用**：空间扩散/传播

## 九、PDE 链（2 条）

### 链39：物理定律→PDE 建立→有限差分/有限元求解
- **结构**：守恒律 → PDE → FDM/FEM → 数值解
- **适用**：热传导/流体/弹性力学

### 链40：数据驱动 PDE→神经网络参数化 PDE→PINN 求解
- **结构**：物理约束 → NN 参数化 PDE → 自动微分 → PINN 求解
- **适用**：已知物理规律但参数未知

---

## 十、图论评价链（3 条）

### 链41：网络构建→中心性分析→关键节点识别
- **结构**：邻接矩阵 → 度/介数/接近/特征向量 → 关键节点排序
- **适用**：复杂网络关键要素识别

### 链42：网络构建→社区检测→模块功能分析
- **结构**：邻接矩阵 → Louvain/InfoMap → 模块角色分析
- **适用**：网络社区结构

### 链43：图嵌入→节点表示学习→下游任务
- **结构**：Node2Vec/GCN → 低维向量 → 下游分类/预测/聚类
- **适用**：大规模图+下游预测/分类

---

## 十一、多阶段耦合链（2 条）

### 链44：子系统独立建模→耦合接口定义→联合求解
- **结构**：各子系统独立 → 耦合变量 → 迭代收敛
- **适用**：多子系统相互影响
- **真题参考**：2022A 波浪能

### 链45：粗粒度初解→细粒度精修→全局一致性检查
- **结构**：简化模型快速求解 → 精细模型局部优化 → 约束回代验证
- **适用**：多阶段递进求解

## 链选择决策树

```
子问题间有无依赖/反馈？
  ├─ 有 → 多阶段耦合链(44/45)
  └─ 无 →
       任务类型？
         ├─ 预测 → 高维？→ 链1 | 非平稳？→ 链2 | 表格型？→ 链3
         ├─ 评价 → 纯数据？→ 链7 | 含定性？→ 链8 | 样本差异大？→ 链9
         ├─ 优化 → 预测前置？→ 16 | 多目标？→ 17 | 非凸？→ 18
         ├─ 分类 → 高维小样本？→ 25 | 标注少？→ 26
         ├─ 仿真 → 链27
         ├─ 时空数据 → 空间插值？→ 28 | 网络结构？→ 29 | 空间异质？→ 30
         ├─ 多源数据 → 可合并？→ 32/33 | 不可合并？→ 35 | 目标域少？→ 34
         ├─ 系统动力学 → 链36/37/38
         ├─ PDE → 参数已知？→ 39 | 参数未知？→ 40
         └─ 图论 → 关键节点？→ 41 | 社区？→ 42 | 下游任务？→ 43
```

## 链质量检查清单

| 检查项 | 判据 |
|--------|------|
| 链首合理性 | 第一步预处理是否与数据特征匹配 |
| 链中信息传递 | 上一步的输出类型是否与下一步输入兼容 |
| 链尾必要性 | 最后一步是否增加了独立价值（不是冗余） |
| 链长适度性 | 3-4 环节最佳，> 5 环节需特殊论证 |
| 替代链对比 | 是否存在更短/更简单的链达到相近效果 |
