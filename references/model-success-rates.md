## 死亡组合（避免使用）

| 模型链 | 近5年国一 | 死亡原因 |
|--------|:--------:|----------|
| 单一灰色预测(样本>50) | 0 | 信息利用率极低，无法捕捉复杂模式 |
| AHP 主观赋权(无组合) | 0 | 一致性常不通过，主观性过强 |
| 简单线性回归(多变量) | 0 | 忽略交互效应，建模深度不足 |
| 纯 ARIMA(无创新/无残差修正) | 1(国二) | 创新性=0，近3年无国一 |
| PSO(无改进) | 1(国二) | 早熟收敛，无改进即无创新 |
| 逻辑回归(无特征工程) | 0 | 线性边界，无法处理复杂分类 |
