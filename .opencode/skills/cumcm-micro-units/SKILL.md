# 微单元覆盖性检查库

## 使用方式

cumcm-writer 写完每节后，调用本 Skill 对应该节的清单，逐句检查：
- ✅ 该句在正文中有对应内容（不要求逐字匹配，意思到了即可）
- ⚠️ 该句内容正文中缺失——补写
- — 该句不适用本题——跳过

---

## 摘要（48 句 / 6 段）

### 第 1 段：问题定性（8 句）
- [ ] 1.1 一句话点明研究领域
- [ ] 1.2 一句话点明问题本质（不是"随着...发展..."）
- [ ] 1.3 一句话说明为什么这个问题难
- [ ] 1.4 一句话说明本文的核心视角/切入点
- [ ] 1.5 一句话说明本文的建模方法论
- [ ] 1.6 一句话说明模型体系（逻辑关系，不是罗列）
- [ ] 1.7 一句话说明创新点
- [ ] 1.8 一句话说明创新带来的改善

### 第 2 段：Q1 处理（6 句）
- [ ] 2.1 Q1 的任务描述
- [ ] 2.2 Q1 采用的模型名称
- [ ] 2.3 Q1 的关键结果（含数字）
- [ ] 2.4 与基线的对比（若有）
- [ ] 2.5 结果的可靠性说明
- [ ] 2.6 Q1 结论的一句话概括

### 第 3 段：Q2 处理（6 句）
- [ ] 3.1 Q2 的任务描述
- [ ] 3.2 Q2 的模型（与 Q1 的关系）
- [ ] 3.3 Q2 的求解方法
- [ ] 3.4 Q2 的关键结果（含数字+对比基准）
- [ ] 3.5 Q2 结果相对于 Q1 的改进/扩展
- [ ] 3.6 Q2 结论的一句话概括

### 第 4 段：Q3 处理（6 句）
- [ ] 4.1 Q3 的任务描述
- [ ] 4.2 Q3 的模型（与前两问的递进关系）
- [ ] 4.3 Q3 的求解方法
- [ ] 4.4 Q3 的关键结果（含数字+对比基准）
- [ ] 4.5 Q3 的跨验证/统计检验结果
- [ ] 4.6 Q3 结论的一句话概括

### 第 5 段：Q4 处理（若有，6 句）
- [ ] 5.1 Q4 的任务描述
- [ ] 5.2 Q4 的全局优化框架
- [ ] 5.3 Q4 的算法设计（含近似/优化策略）
- [ ] 5.4 Q4 的关键结果（含数字）
- [ ] 5.5 Q4 的敏感性分析结论
- [ ] 5.6 Q4 结论的一句话概括

### 第 6 段：总结（4 句）
- [ ] 6.1 全文方法论的总结
- [ ] 6.2 最终结论的工程/决策意义
- [ ] 6.3 模型的推广价值
- [ ] 6.4 关键词（3-8 个，用分号分隔）

## 问题重述与分析（12 句）

### 背景说明（4 句）
- [ ] 研究领域/场景的一句话介绍
- [ ] 该场景中存在的核心挑战
- [ ] 已有方法的不足
- [ ] 本文要解决的具体问题

### 题目任务分解（4 句）
- [ ] Q1 任务的精确复述（引用原文）
- [ ] Q2 任务的精确复述
- [ ] Q3 任务的精确复述
- [ ] Q4 任务的精确复述（若有）

### 本文思路概述（4 句）
- [ ] 本文的建模方法论
- [ ] 各问之间的递进关系
- [ ] 核心创新一句话
- [ ] 预期达到的目标

## 模型假设（8 句 + 符号表）

### 每条假设 × N（每假设 1 句合理性论证 + 1 句影响分析）
- [ ] 假设的精确陈述
- [ ] 该假设的合理性论据（数据支撑/常识/题目建议）
- [ ] 该假设的放宽对结论的影响预估

### 符号说明（表格）
- [ ] 全文所有符号在表格中汇总
- [ ] 每个符号含含义和单位
- [ ] 符号命名体现核心概念

## 模型建立与求解（核心章节，按 Q 核查）

### 每个子问题 ×（8 句）

#### 建模思路（1 句）
- [ ] 用朴素语言说明本问的核心建模思想

#### 变量定义（2 句）
- [ ] 决策变量/输入变量定义及取值范围
- [ ] 参数定义及标定方法

#### 公式推导（4 句）
- [ ] 标准/基础公式
- [ ] 标准公式在本问的不足
- [ ] 创新/改进后的公式
- [ ] 改进公式的"这意味着..."解释

#### 求解算法（3 句）
- [ ] 算法选择的理由
- [ ] 算法流程简述（3-5 步）
- [ ] 收敛性/复杂度分析

#### 结果展示（4 句）
- [ ] 关键数值结果（三线表格式）
- [ ] 与基线的对比
- [ ] 图表说明（每张图标配一句"图X说明..."）
- [ ] 敏感性分析

#### 结果分析（3 句）
- [ ] 结果说明了什么（物理/经济/工程直觉）
- [ ] 差异来源分析（哪里好，哪里差，为什么）
- [ ] "这一结果对实际决策意味着..."

---

## 模型评价（12 句）

### 优点（3 句）
- [ ] 优点 1 + 量化证据
- [ ] 优点 2 + 量化证据
- [ ] 优点 3 + 量化证据

### 不足（4 句）
- [ ] P3 预判了什么
- [ ] P4 实测验证/证伪了什么
- [ ] 量化边界：在什么条件下退化，退化多少
- [ ] 原因分析：为什么退化

### 改进方向（3 句）
- [ ] 改进方案 1 + 需要什么额外条件
- [ ] 改进方案 2 + 需要什么额外条件
- [ ] 改进的预期效果估计

### 推广价值（2 句）
- [ ] 本文方法的可迁移场景
- [ ] 方法对同类问题的启示
