# CUMCM 学术调色板

## 一、5套场景化配色

### 1. contrast_3（通用对比，2-3组）
```python
["#2878B5", "#9AC9DB", "#C82423"]
#  深蓝       浅蓝       深红
```
适用：折线图、柱状图的多模型/多方案对比。深蓝=我方方法，深红=基线/对比。

### 2. gradient_6（多组渐变，4-6组）
```python
["#8ECFC9", "#FFBE7A", "#FA7F6F", "#82B0D2", "#BEB8DC", "#E7DAD2"]
#  青绿       杏橙       珊瑚       钢蓝       薄紫       米灰
```
适用：堆叠面积图、多类别热力图。低饱和渐变，视觉柔和，信息层级自然。

### 3. directional（正负标注）
```python
{"positive": "#2E7D32", "negative": "#C62828", "neutral": "#78909C"}
#            深绿             深红             蓝灰
```
适用：瀑布图、偏差图、残差图的正负方向标注。仅用于方向，不用于分类配色。

### 4. spatial（地理/空间）
```python
"viridis"
```
适用：热力图、等高线图、三维曲面。matplotlib 内置，感知均匀，色盲安全。

### 5. pastel_family（同色系深浅，多细分对比）
```python
["#1565C0", "#1976D2", "#1E88E5", "#42A5F5", "#90CAF9"]
#  最深        深         中         浅         最浅
```
适用：分段条形图、多分组柱状图（同类方法不同参数/不同时期）。同一色系不同深浅表示同一类事物的不同变体。

### hero_highlight（主次对比，排名图用）
```python
{"hero": "#1565C0", "compare": "#FF8F00", "other": "#B0BEC5"}
#        深蓝(最优)       琥珀(对比)         浅灰(其余)
```
适用：排名图——最优项深蓝突出，对比项琥珀，其余灰色弱化。

---

## 二、使用规则

1. **每张图 ≤4 种颜色**。出现第4种颜色时必须有明确语义
2. **hero 色（深蓝）用于突出最优/我方方法**，全论文不同图中同一含义保持同一颜色
3. **绿/红仅用于方向标注**（涨/跌、正/负、通过/不通过），不作为分类色
4. **多类别对比用 pastel_family 同色系深浅**，避免五颜六色的杂乱感
5. **生成后执行灰度测试**：`is_grayscale_distinguishable(colors)` 返回 True 才通过
