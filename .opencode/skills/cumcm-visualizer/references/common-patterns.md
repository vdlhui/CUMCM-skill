# CUMCM 常用布局模式

## 模式 1：分组柱状图（Grouped Bar）
- 布局：等宽分组柱 + 误差线 + 最优柱深色突出
- 标注：最优✅标记 + 百分比差距 + 盈亏平衡虚线
- 配色：hero_highlight（最优深蓝，其余浅灰）或 contrast_3

## 模式 2：水平排名图（Ranked Bar）
- 布局：水平柱 + 误差线 + 柱端数值标注
- 最上方为最优，第一柱深色，其余同色系渐浅
- 配色：pastel_family（最优最深色）

## 模式 3：趋势折线+置信带（Trend Line）
- 布局：折线 + 半透明置信带 + 标记点
- 多组对比用不同颜色折线 + 各自置信带
- 关键拐点/平台期标注（箭头+文字）

## 模式 4：散点+回归+ρ标注（Scatter Corr）
- 布局：散点 + 回归线 + 半透明置信带
- 图内文本框标注 Spearman ρ 值和 p 值
- 配色：点用 hexbin 密度着色或单色半透明

## 模式 5：双面板敏感性（Sensitivity Dual）
- 布局：左右双面板 (a)(b)，共享 y 轴标签
- (a) M扫描：折线+标记点，标注饱和趋势
- (b) T扫描：折线+标记点，标注饱和点
- 配色：同一色系，两面板用不同深浅

## 模式 6：差距标注柱状图（Gap Annotated）
- 布局：分组柱 + 柱间差距标注
- 适用：DP反例贪心vs最优、烧蚀实验、多方案优劣对比
- 标注：百分比差距 + 箭头 + 纹理/斜线填充强调差距

## 模式 7：双y轴组合（Line+Bar）
- 布局：左y轴（折线-趋势） + 右y轴（柱状-绝对值）
- 配色：折线和柱状用不同色系，图表标题标注双轴含义

## 模式 8：热力图+行/列边际趋势（Heatmap+Marginal）
- 布局：中心热力图 + 上方/右侧边际折线
- 注释：关键格子标注数值 + 最优/最差行列高亮边框

## 图例优化
- 单组数据：不显示图例（用标题说明）
- 2-4组数据：图例放在图内右上角或图外右侧
- ≥5组数据：考虑用 direct label（直接标注在线上/柱旁）替代图例
