# CUMCM 视觉立场

## 一、字体策略
- 自动检测系统中文字体：SimHei > Microsoft YaHei > SimSun > 降级英文
- 全论文图表字号：标题 10pt，轴标签 9pt，刻度 8pt，图例 8pt
- 中文字体为无衬线（sans-serif），英文/数字默认匹配

## 二、坐标轴策略
- 右侧/顶部脊线永远关闭
- 仅保留左（y）和底（x）脊线
- 默认无网格线；仅在热力图/极坐标图中保留参考网格
- 坐标轴标题含物理量+单位（如"净收益 / 洛克贝"）

## 三、配色策略
- 每张图 ≤4 种颜色
- 最优策略/我方方法 → 深蓝 hero 色
- 基线/对比方法 → 琥珀 compare 色
- 正向/改善 → 深绿 positive 色
- 负向/亏损 → 深红 negative 色
- 多类别对比用单色系不同透明度（0.3→1.0）
- 绿/红仅用于方向标注，不用于分类
- 所有配色需通过灰度打印验证（转灰度后各组仍可区分）

## 四、误差与统计
- 凡有均值，必带误差（95%CI 或 SE）
- 误差线颜色比柱体/线体深一档
- 组间显著性对比：括号 + p 值星号标注（***/**/*/n.s.）
- 临界值/阈值用灰色虚线标注

## 五、标注策略
- 最优值：柱顶/点上方标注"✅ 最优" 或数值
- 百分比差距：柱间或点间用短箭头 + 百分比
- 关键洞察：椭圆或圆角矩形文本框，直接告诉评委"看这里"
- 图例优先放在图内空白区（左上/右上），其次图外右侧

## 六、尺寸一致性
- 全论文所有图统一宽度：单栏 8cm 或双栏 16cm
- 高度根据内容自适应，但同一类型图的高度应一致
- dpi = 300（PNG），PDF 为矢量无 dpi 概念

## 七、导出策略
- 主格式：PNG（300DPI，RGB，用于 Markdown/Word 嵌入）
- 辅格式：PDF（矢量，用于 LaTeX 嵌入）
- `bbox_inches='tight'` 裁剪白边
- 文件名格式：`fig_{序号}_{简短描述}.png/pdf`

## 八、反冗余
- 多面板图的每子图必须回答不同问题
- 三层次递进：概览（overview）→ 差异（deviation）→ 关系（relationship）
- 同一数据不在多张图中重复出现

## 九、色盲友好
- 重要信息不单靠颜色区分（配合线型/标记/标注文字）
- 避免红-绿、蓝-黄组合
- 生成后通过灰度转换检查可区分性
