# P4 求解结果验证清单

## 量纲一致性检查（执行 dimensional-check.py）

- [ ] 所有数值变量的量纲与 P0 输出单位体系一致
- [ ] 求和/连乘/积分/微商运算前后量纲匹配
- [ ] 无量纲化处理后结果是否已正确还原量纲

## 边界条件验证（执行 boundary-check.py）

- [ ] 所有约束的上下界均被测试
- [ ] 在约束边界上的解是否合法（不超出/不矛盾）
- [ ] 极端条件（0 / ±∞ / 极值）下模型行为是否合理
- [ ] 负库存/负距离/概率>1/温度<-273°C 等物理不可能值是否已排除

## 数值稳定性

- [ ] 多次运行（seed 0,1,2,3,4）的指标标准差 < 均值 × 20%
- [ ] 无 NaN / Inf 输出
- [ ] 算法收敛曲线无震荡/发散迹象

## 结果合理性

- [ ] 关键数值在题目给定的合理范围内
- [ ] 预测值序列无退化（如全为均值）
- [ ] 聚类结果有无意义簇（全样本一簇或每样本一簇）
- [ ] 优化解是否满足所有显式约束

## 素材包完整性

- [ ] model-results.json 所有字段已填充
- [ ] metrics.json 含基线对比数据
- [ ] table-index.json 每张表有 caption 和 key_finding
- [ ] 敏感性分析数据已收集（关键参数+变化范围+影响量化）
- [ ] 公式编号与 P2 model-innovation.json 一致
